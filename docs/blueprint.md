# **App Name**: ShuttleScore

## Core Features:

- Player Management: Add, edit, and manage badminton player profiles including skill level, availability, age, gender, and name.
- Court Overview: Display an overview of each court with ongoing or scheduled matches.
- Random Match Pairing: Randomly select 4 players for a match, ensuring a maximum of 2 players repeat from previous matches in the day, while balancing the overall skill level of the teams using an AI tool.
- Manual Match Pairing: Allow manual selection of players for a match.
- Match Scheduling: Schedule matches on specific courts and times.
- Match Status Updates: Update the status of the matches (e.g. scheduled, in progress, completed).
- Match Results: Enter and display results of completed matches.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a professional and focused feel.
- Background color: Very light lavender (#F0E6EF) for a calming backdrop.
- Accent color: Violet (#8F00FF) for highlights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and objective look.
- Use clear, minimalist icons for navigation and actions (e.g., player, court, schedule).
- Use a grid-based layout to organize player data and match schedules efficiently.
- Subtle animations for transitions between screens and match updates to enhance user experience.
